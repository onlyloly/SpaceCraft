
# SpaceCraft

SpaceCraft is an interactive 3D room planning application that allows users to design interior layouts in real time.

## Features

* Create rooms with custom dimensions
* Interactive 3D visualization
* Add and arrange furniture
* Add interior objects and decorations
* Configure doors and windows
* Resize and rotate objects
* Switch between 3D and top-down view
* Real-time room area calculation
* Project saving and exporting
* Responsive modern UI

## Technology Stack

* React
* TypeScript
* Three.js
* React Three Fiber
* Drei
* Zustand
* Tailwind CSS
* Framer Motion
* Vite
